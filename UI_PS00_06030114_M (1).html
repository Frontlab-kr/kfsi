자격증
  • 검색되는 자격증 외 등록은 불가합니다.
  • ‘ 직무 관련 자격 종류 ’ 에 해당하는 최종 합격에 대한 자격만 입력
    • → ‘ 자격 ’ 은 상위자격 최대 N개까지 서류전형에서 점수 부여(필기시험, 면접시험 점수 없음) [점수가 인정되는 자격 목록 확인]
  • 자격증은 지원자가 취득 보유한 자격 중에서 4개 까지만 입력 가능(상위자격 순으로 입력).
  • ‘ 자격(면허)번호 ‘ 항목 입력시 ‘ 숫자 0 ’ 과 ‘ 알파벳 O ’ 입력 구분 유의 (한국산업인력공단 자격번호는 숫자 11자리와 알파벳 1자리로 구성)
